# Adaptive_Point_Cloud_Clustering
Here is the implementation of point cloud clustering for Livox HAP Lidar

## How to build ##
```
$ cd ~/catkin_ws/src/
$ git clone https://github.com/yzrobot/adaptive_clustering.git
$ cd ~/catkin_ws
$ catkin_make
```
